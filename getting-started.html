Getting started

Writerside is a plugin for IntelliJ-based IDEs that let you to write, structure, preview, build, and test documentation.

This section will help you get started and show how to do basic things with Writerside.

You will learn how to install the plugin, start a documentation project, create a topic from template, and preview it.

Install the plugin

  1. Go to the plugin page at plugins.jetbrains.com.

  2. Click Get and download a ZIP archive.

  3. Open Settings/Preferences | Plugins.

  4. Click button in the top-right corner and then select Install Plugin from Disk….

  5. Select the archive and click OK.

After you have installed Writerside, you need to either create a new documentation project or a documentation module inside an existing project with a source code. Let's go with the second option.

Add documentation to your dev project

  1. In the Writerside tool window, click or press Alt+Insert.

  2. Specify an instance name and ID.

  3. (Optional) Specify a web path an instance help will be available by and version if you plan to maintain a few versions.

  4. Deselect the Allow search engine indexing option if you don't want your content indexed by external search engines.

  5. Click OK.

A new instance will appear in the toolbar with the ID in parentheses, a new record in a project.ihp file will appear, and a new .tree file will be created.

Create a topic from template

To set up your plugin's minimum viable documentation in minutes, create a few topics from ready-to-use templates.

  1. In the TOC area of the Writerside tool window, select the folder to which you want to add the new topic and click on the toolbar or press Alt+Insert.

  2. Click Topic from Template and choose from a built-in template for different types of topics: How-to, Overview, Reference, or Tutorial.

    You can configure your own custom templates based on the predefined ones or tailored to the types of articles you often write.

  3. In the New Topic dialog, specify the topic title and the file name (you can change the title later if needed, see Edit titles, filenames and IDs).

    Writerside uses the file name to automatically generate the topic ID, so it must be unique within your project and meaningful so that you can easily reference it from other topics.

The newly created topic will appear in the TOC view and will open in the editor.

Start editing the template and building content — the original template will not be affected.

Override the template's content with your own, change the image to yours to illustrate the process. You may want to add the dark version of the image, read how to do it here.
