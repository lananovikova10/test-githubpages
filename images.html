Images

Here's how you insert an image in Markdown: ![alt text](image.png "title").

Enhance the image by placing the following attributes in the curly brackets after the block:

  • width

  • border-effect

Specify an image width

Using the width attribute you can specify the width of the image. Its height will also be changed proportionally.

Markup

Result

![](sample-image-white.png) {width="250px"}

Change an image border style

To add a border to an image and specify its appearance, use the border-effect attribute with a line or rounded as possible values.

Markup

Result

![](sample-image-white.png) {border-effect="line"}
