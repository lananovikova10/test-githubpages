Tables

To create the table, use a table tag. A table consists of tr elements for rows and td elements for cells.

The table has the following attributes:

style

[header-row|header-column|both|none] Define the table header appearance. The header is an element used to label each column, the text inside it is marked bold.

The table cell (td tag) has the following attributes:

colspan

Define the number of columns a cell should span.

rowspan

Specify the number of rows a cell should span.

sorted

[asc|none|desc] Sort the column data.

width

Set the column width in pixels or percent.
