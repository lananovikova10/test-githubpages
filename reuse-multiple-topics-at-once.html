Reuse multiple topics at once

You can reuse a single snippet and multiple topics at once from a table of contents.

For example, if the installation and setup flow is applicable for multiple instances.

Make a table of contents section reusable

Create a TOC library which is the same as an instance, but is used to store the toc-elements for reuse.

The TOC library contains topics grouped in snippets that can be reused in other products.

  1. In the Writerside tool window, click or press Alt+Insert.

  2. Specify a library name and ID.

  3. Click OK.

  4. In the TOC library .tree file group toc-elements under a snippet element and assign an id.

    <snippet id="installation-guide"> <toc-element topic="Install.topic"/> <toc-element topic="Setup.topic" toc-title="Initial setup"/> <toc-element topic="Update.topic"/> <toc-element topic="Uninstall.topic"/> </snippet>

  5. Another option is to create a wrapper topic that includes all the needed topics and assign an id attribute to it.

<toc-element toc-title="Installation_guide.topic" id="installation-guide"> <toc-element topic="Install.topic"/> <toc-element topic="Setup.topic" toc-title="Initial setup"/> <toc-element topic="Update.topic"/> <toc-element topic="Uninstall.topic"/> </toc-element>

Reuse multiple topics at once

You can reuse both toc-elements and snippets from TOC library by their ids.

To reuse the tree section:

  1. Add the include tag to a tree.

  2. In the from attribute specify an instance id you want to include a wrapper or a snippet from, in this case, it is toc_chunks.

  3. In the target attribute specify the element id.
