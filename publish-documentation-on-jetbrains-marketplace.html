Publish documentation on JetBrains Marketplace

Writerside allows you to publish documentation on the separate Documentation tab on the plugin page on JetBrains Marketplace instead of taking the user to an external page or using a custom page.

  1. Place the documentation project in your plugin's repository or in a separate one.

  2. Build the help instance locally to produce a ZIP archive.

  3. Send your plugin's ID (5 digits) and a ZIP archive name to our team.

  4. Our team will create the TeamCity build configuration and send the link to you.

  5. You will be able to build the documentation, then publish it to staging to check.

  6. The documentation will appear in the Custom pages section of the plugins' page.

    Click Preview to see how users will see it.

  7. Go back to the TeamCity and promote the docs to production.
