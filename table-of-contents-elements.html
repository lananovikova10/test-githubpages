Table of contents elements

The toc-element element has the following attributes:

topic

Specify the topic filename.

toc-title

Specify the topics' title. This title will appear in the table of contents and will override the title specified in the topic. If you specify the toc-title, but not the topic it will create a wrapper topic with no content in it.

hidden

Set to true to hide the topic from a table of contents. The topic is accessible by direct links to it.

sort-children

[none|ascending|descending] Specify the order to sort the topics' children. By default there is no sorting, topics appear in an order they are placed in .tree file.

accepts-web-file-names

Specify the HTML file name of the removed page.

accepts-web-file-names-ref

Specify the rule id from the redirection-rules.xml.

target-for-accepts-web-file-names

Specify the full URL where to redirect a user.
