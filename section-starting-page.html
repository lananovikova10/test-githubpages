Starting page

Starting pages are an alternative to wrapper topics. They provide design and markup for highlighting the most useful and popular topics of a section or the whole help.

To make a page starting for the instance, right click the topic's title in the TOC area of the Writerside tool window and select Set as Starting Page. It will appear in the start-page attribute of the instances' .tree file.

For consistency reasons, starting pages have a fixed layout. For predefined groups, you can change the amount of cards and their content. You can also add custom card and link groups and adjust their width.

To create a starting page, create a topic and add the section-starting-page tag as a child of the topic tag.

If the section-starting-page tag is present on a page, all other content elements are ignored.

All links on a starting page must either point to topics with a summary or have the summary attribute.

Page structure

The section-starting-page content structure can look like this:

<section-starting-page> <title>Section starting page title</title> <description> Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed condimentum feugiat sagittis. </description> <spotlight> <a href="Topic.topic" description="Sed condimentum feugiat sagittis" type="start"/> ... </spotlight> <primary> <title>Main group</title> <a href="Topic.topic" description="Sed condimentum feugiat sagittis"/> ... </primary> <secondary> <title>Highlighted group</title> <a href="Topic.topic" description="Sed condimentum feugiat sagittis"/> ... </secondary> <misc> <cards> <title>Custom cards</title> <a href="Topic.topic" description="Sed condimentum feugiat sagittis"/> ... </cards> <links display="wide"> <![CDATA[> <group> <title>Custom links</title> <a href="Topic.topic" description="Sed condimentum feugiat sagittis"/> ... </group> ... </links> </misc> </section-starting-page>

primary and secondary groups are mandatory, spotlight and misc are optional.

Groups

Links on a starting page have to be divided into groups.

Use the spotlight tag to highlight the most useful and popular topics of a section. The type attribute defines the card's icon.

Place other important topics in the primary and secondary groups. These groups are mandatory.

If necessary, use the custom-groups section. Follow the procedure below to create a group.

Add a group

  1. Under the misc tag, add either cards (to have a few square cards) or links tag (to have a group of links).

  2. Add the title tag or attribute.

  3. Add links to the group.

  4. (Optional) Change the columns' width.

    Add the display attribute and change its value. By default, there are three columns with a normal display type. If you need two columns, set the value to wide.

  5. (Optional) If you used links tag, wrap each link set with the group tag and add a title for each of them.
