Find Reusable Content

You can locate the content what worth to be reused using IntelliJ IDE's feature called Find in Files.

First, try to find reusable content that you can add to the library. For example, if you describe some installation or setup procedure and heavily use the control names, check where these controls and paths are also mentioned. To do this:

  1. Highlight the control or ui-path element.

  2. Right-click and select Find in Files or press Ctrl+Shift+F

  3. If the content is repeated two or more times, consider moving it into a reusable snippet.
