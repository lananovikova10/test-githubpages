Markdown Markup

Writerside uses the CommonMark specification with the flexmark as a processor.

All text formatting options (bold, emphasis, inline code), as well as images, links, tables, code blocks, lists, and headings are supported.

On top of that, it allows to extend some markup elements using custom attributes or inject semantic markup blocks into Markdown.

This section highlights some of the special Markdown features you can use in your docs.

Also, you may use the IntelliJ IDEs bundled Markdown plugin features, as:

  • Insert actions by pressing Alt+Insert (for images, tables, and links);

  • Drag and drop for images and internal links;

  • Live Table edit;

  • Formatting actions in the floating toolbar.
